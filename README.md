# depth_inpainting

The idea is to convert 2D photos into 3D using inpainting techniques.

# Notebook

[Photo to 3D](https://github.com/vornitier/depth_inpainting/blob/main/photo_to_3d.ipynb) Little bit of Midas for depth and some zooming.

## Installation

- Open Google Colab. Install packages. Run the notebook.
